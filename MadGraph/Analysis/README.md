# To run code


```
root.exe -b 
.L HiggsAnalysis_14TeV.C
HiggsAnalysis_14TeV t
t.GetEntry(-1); 
t.Loop();
```
or

```
root -b RunMe.cxx++
```
it came from nice discussion here: `https://root-forum.cern.ch/t/compile-and-run-from-command-prompt/13799/7`

```
.L HiggsDistribution.C
HiggsDistribution t
t.GetEntry(-1)
t.Loop()
hHiggsPt->Draw();
```
